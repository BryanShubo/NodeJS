##1. CMD command:

>**node --version**

// return version

>**node**  

// Go to node

>**.help** 

// show repl options

>**.break**

// this gets you out

>**.clear**

// alias for .break

>**.exit**

// exit the repl

>**.load**

// load js from a file into the repl session

>**.save**

// save all eveluated commands in this repl session to a file

>**control + D**

// gets out of node
